# Puzzle Escape Room

## Game Concept
A first-person puzzle escape room where players must solve 5 different puzzles within 3 minutes to unlock the exit door. Complete all puzzles before time runs out to win!

## Puzzle Tasks
1. **ATM Puzzle** - Interact with the ATM to solve the puzzle
2. **Cube Puzzle** - Manipulate cubes to complete the challenge
3. **Puzzle 3** - Complete the third puzzle mechanism
4. **Puzzle 4** - Solve the fourth puzzle challenge
5. **Pressure Plate** - Activate the pressure plate to complete the final puzzle

## Asset Sources
- **Starter Assets**: Unity Starter Assets (First Person Controller)
- **ATM Model**: [CGY_ATM](https://assetstore.unity.com/)
- **Door Model**: [double-door](https://assetstore.unity.com/)
- **Wall & Floor Asset**: [_Barking_Dog](https://assetstore.unity.com/)

## Controls
- **WASD** - Move
- **Mouse** - Look around
- **E** - Interact with puzzles
- **Space** - Jump
- **Shift** - Sprint

## Win/Lose Conditions
- **Win**: Complete all 5 puzzles before the timer runs out
- **Lose**: Timer reaches 0:00 before completing all puzzles
